Minha Biblioteca

Sobre

API desenvolvida para integrar a biblioteca online com qualquer sistema.

Os endpoints recebem a requisição com as informações necessárias no body.

A API processa os dados e envia para a biblioteca online verificar.

Caso a biblioteca retorne positivo, a URL autenticada é devolvida ao sistema requerente.

Construção

Criados endpoints separados UNEF e UNIFAN

A API conversa com a API da Biblioteca Virtual

Necessária SECRET KEY no header da requisição.
