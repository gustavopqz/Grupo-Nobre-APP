O que é o netlify:

Netlify é uma plataforma de hospedagem https que contém planos grátis.

Alterações no projeto:

1 - É preferível usar o cmd do windows diretamente do que pelo vscode.

2 - No CMD deve-se acessar o patch do projeto. Neste localizado em Documents > Gustavo > GRUPO NOBRE APP.

3 - Somente arquivos da pasta "public" podem ser usados no frontend.

4 - Todo o backend é formado na pasta "functions".

5 - Usar o comando help para ter acesso às funções do pacote. "netlify help" ou "ntl help".

6 - Comando mais usados:

  • 6.1 - "ntl dev" para abrir o abiente de desenvolvimento em localhost.
  • 6.2 - "ntl functions:list" para listar as funções existentes (endpoints) e se estão em produção.
  • 6.3 - "ntl deploy" para dar deploy. O netlify mostra um link teste antes de subir realmente para a produção para verificar se tudo está correto.

7 - Para mais informações acesse cli.netlify.com.

Apresentação

Esta API recebe dados do CANVAS UNEF e CANVAS UNIFAN e retorna uma URL autenticada pela Minha Biblioteca (empresa de biblioteca virtual).

A URL autenticada só é retornada se o RA do aluno estiver registrado na nossa base de dados.

Entre os dados recebidos estão:

  • Primeiro Nome
  • Último Nome
  • RA do aluno

Endpoints

.netlify/functions/bibli_unef

.netlify/functions/bibli_fan

Requisição

method: post.

body: JSON.stringify({
firstName: string,
lastName: string,
ra: string })
