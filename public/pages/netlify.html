NETLIFY

Sobre

O Netlify é uma plataforma de hospedagem https que contém planos grátis.

Uma mesma conta Netlify consegue hospedar mais de um projeto.

O deploy das aplicações pode ser feito todo via prompt de comando.

Os comandos no CMD podem ser dados por "netlify" ou "ntl".

Construção

1 - Acesse o patch do projeto.

2 - Verifique a versão do Netlify por "ntl --version" ou instale-o Netlify por "npm install netlify-cli" -g

2.1 - Instale os pacotes necessários com npm.

3 - Crie o arquivo netlify.toml na raiz do projeto passando:
[build]
functions = "functions"
publish = "public"

4 - Crie uma pasta "functions" para construção do backend da aplicação.

4.1 - Os endpoints da API são acessados por meio de "/.netlify/function/nome_da_funcao".

5 - Crie uma pasta "public" para construção do frontend da aplicação.

Aplicação

1 - Verifique se o netlify-cli está instalado na máquina por "netlify --version", caso não, instale-o: "npm install netlify-cli -g".

2 - "ntl help" revela todos os comandos do netlify.

3 - "ntl "command" help mostra como usar este determinado comando.

4 - Mais usados:
a) "ntl init" - Incializa o projeto.
a) "ntl login" - Conecta o projeto à conta Netlify.
b) "ntl dev" - Gera um ambiente de desenvolvimento.
c) "ntl env" - Manipulação das variáveis de ambiente.
d) "ntl status" - Verifica o e-mail conectado, a url do deploy, etc.
e) "ntl functions" - Informa o status dos endpoints.
f) "ntl deploy" - Verifica se o deploy será feito corretamente.
g) "ntl deploy --prod" - Sobe o projeto para a web.
